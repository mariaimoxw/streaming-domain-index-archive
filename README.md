# streaming-domain-index-archive
该仓库用于整理流媒体影视站点的域名信息与页面结构特征，通过文字方式记录分类逻辑、筛选规则及标签分布情况，便于进行跨站点结构对比与长期归档。项目不提供任何影视播放资源，仅保留公开页面层面的信息元素。所有资料来源于公开网络环境，仅供信息汇总与研究参考，不代表任何形式的推荐或推广行为。
